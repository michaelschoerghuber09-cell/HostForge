# HostForge

HostForge is a desktop application designed to simplify the creation and management of Minecraft servers.

Instead of manually downloading files, configuring settings, and managing different server types, HostForge provides a simple interface to set up and manage Minecraft servers more easily.

## Goal

The goal of HostForge is to create an all in one server management tool that makes Minecraft server creation faster, and easier.

## Tech Stack

- **Rust:** Backend, system operations, server management
- **Tauri:** Desktop application framework
- **React TypeScript:** – User interface
- **Git:** – Version control

## Planned Features

- Minecraft server setup wizard
- Server management dashboard
- Automatic server file handling (eg. accepting eula)
- Configuration management
- Plugin and mod support
- Player and whitelist management
- Backup and monitoring tools

## About the Project

The idea for this project randomly came to my mind when I was thinking about repetitive tasks that are annoying to do manually.

After some research, I found that there are already similar projects that solve parts of this problem. One of them has many similarities to the idea I had, so I want to give credit and mention it as inspiration.
